\# Factory-to-Customer Shipping Route Efficiency Analysis



This project analyzes factory-to-customer shipping routes for Nassau Candy Distributor to identify efficient routes, delay-prone routes, and regional logistics bottlenecks.



\## Objectives

\- Measure shipping lead time performance

\- Benchmark route efficiency

\- Identify geographic bottlenecks

\- Compare shipping performance across ship modes

\- Build an interactive logistics intelligence dashboard



\## Key Features

\- Route efficiency leaderboard

\- Regional bottleneck analysis

\- Ship mode performance comparison

\- KPI monitoring system

\- Interactive Streamlit dashboard



\## Tech Stack

Python, Pandas, NumPy, Streamlit, Plotly, SQL, Jupyter Notebook
